## Installation

1. Clone the repository:

```bash
git clone https://github.com/Sran012/Probex.git
```
2. Navigate into the project folder:
```bash
cd Probex/
cd client/
```

3. Install dependencies:
```bash
npm Install
```

4. Start the application:
```bash 
npm run dev
```
